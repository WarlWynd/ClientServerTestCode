# Running the Game Client

## Requirements
- Java 23+ installed on your machine
  Download from: https://www.oracle.com/java/technologies/downloads/

## How to Run

### Windows (PowerShell or Command Prompt)
```
gradlew.bat :client:run
```

### Mac / Linux (Terminal)
```
./gradlew :client:run
```

## First Time Setup
The first run will download Gradle automatically — this requires an internet connection
and may take a minute or two. Subsequent runs will be instant.

## Server
The game server is running at 192.168.1.153:9876
Make sure you are on the same network (192.168.1.x) to connect.

## Usage
1. Launch the client using the command above
2. Register a new account on the login screen
3. Log in and start playing!
